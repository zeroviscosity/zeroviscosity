Profile

Hi! My name is Kent and I am a full stack web developer based in Toronto, Ontario. My focus is on building applications that let data flow smoothly back and forth between the server and the user.

I'm comfortable working both server-side and client-side. On the server I tend to use open source software such as PHP, Python, Ruby or Node.js. On the front-end I like to use as much HTML5 and CSS3 as possible, along with JavaScript and jQuery. Ultimately I believe in using the right tool for the job.
